# AWS S3 Static Website Hosting

## Project Description
This is a mini cloud computing project where a static website is hosted using Amazon S3.

## Services Used
- Amazon S3
- AWS Free Tier

## Features
- Serverless website hosting
- Public access using bucket policy
- Low cost and highly available
